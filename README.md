# Brian Story

React + TypeScript + Vite with Material UI and SCSS.

## URLs

- Homepage: https://brian00uni.github.io/story
- Local API: http://localhost:3000

## Requirements

- Node.js
- Yarn (classic)

## Install

```bash
yarn
```

## Run

Frontend + API server:

```bash
yarn dev
```

API server only:

```bash
yarn server
```

## API Notes

- `/api/board` is currently in-memory only.
- Data resets when the server restarts.

## Formatting

Prettier scripts:

```bash
yarn format
yarn format:check
```
